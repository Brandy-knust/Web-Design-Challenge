Summary: Lattitude vs. X

Visualizations

In this activity, we generated over 500 random Lattitude and Longitude points and then found cities that were at those points. We then analyzed weather data found at those points for Temperature, Humidity, Cloudiness, and Wind Speed. We were able to see if the Lattitude, or distance from the Equator, affected each of these conditions. We plotted each of the points against the Lattitude to see if there was a correlation.

In addition, we ran statistical analyses to see the relationships between each of the different data sets. On this site, you will find just how correlated each variable is to the Lattitude. You will see that, other than Temperature, many of the variables have very little to do with the actual Lattitude, but more with other factors.
